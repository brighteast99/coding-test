# 코테 풀이 모음

## 테스트케이스 실행
```sh
node index.js test <solutionPath> -s -r
# -s: 출력 무시. optional.
# -r: 개별 결과 무시 (전체 결과만 출력). optional.
```
 
## 문제 템플릿 생성
```sh
node index.js create <problem-name> -f <problem-from> -c <category>
# -f: 문제 출처 (e.g. programmers, BOJ, ...). optional.
# -c: 문제 분류 (e.g. 탐색, 정렬, ...). optional.
```
